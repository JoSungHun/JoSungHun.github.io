Webhacking.kr 6번 문제

6번 문제 소스코드는 다음과 같다.

user 쿠키가 없다면 $val_id=”guest”, $val_pw=”123qwe”로 값을 주고 base64로 20번 인코딩한 후, 1은 !, 2는 @, 3은 $, 4는 ^, 5는 &, 6은 *, 7은 (, 8은 )로 변환하고 각각 user, password 쿠키에 값을 넣는다.

그 후 각각에 맞게 다시 숫자로 변환해 준 후 base64로 20번 디코딩한다. 디코딩과정을 거친후 id가 “admin”, pw가 “nimda”이면 solve함수가 호출된다.
admin, nimda값을 base64로 20번 인코딩한 후, 숫자를 위와 같이 바꿔서 쿠키에 입력하면 문제가 풀릴것으로 보인다.

base64를 20번이나 돌리고 숫자를 하나하나 바꿔주기 힘드므로 파이썬으로 간단하게 코딩을 해서 문제를 해결해 본다.

파이썬을 돌린결과를 쿠키에 입력하면 문제가 풀린다.

Webhacking.kr 1번 문제

Webhacking.kr 1번문제

아래쪽 php 코드를 보면 user_lv 쿠키 값이 5보다 크고 6이상이면 1로 값을 바꾸고, 5보다 크면 solve 함수가 호출된다. 따라서 5보다크고 6보다 작은 숫자로 user_lv 쿠키값을 변경해 주면 문제가 풀린다.

크롬 확장프로그램인 EditThisCookie를 사용하여 쿠키값을 변경해주면 문제가 풀린다.
